# **App Name**: FinTrack Lite

## Core Features:

- Income Tracking: Allows users to input their income from one or more sources.
- Expense Tracking: Allows users to input recurring expenses like utilities and monthly credit card bills. 
- Balance Visualization: Dynamically displays the total income versus total expenses, highlighting the difference in green (positive) or red (negative).

## Style Guidelines:

- Primary color: Black or dark grey for the background.
- Secondary color: Yellow for text and important UI elements.
- Color: Red for expenses/negative balances.
- Color: Green for income/positive balances.
- Accent: Blue (#29ABE2) for interactive elements and highlights.
- App name prominently displayed in the top-left corner.
- Clean and intuitive layout for easy navigation.
